# Conversor de Unidades 🧮

Um conversor de unidades simples e interativo entre **milhas e quilômetros**.

---

## Funcionalidades

- Converter **milhas para quilômetros**  
- Converter **quilômetros para milhas**  
- Resultados arredondados com **2 casas decimais**  
- Interface simples e intuitiva

## Como Usar

1. Abra o arquivo `index.html` no navegador.  
2. Digite o valor que deseja converter.  
3. Escolha a unidade (milhas ↔ km).  
4. Clique em **Convert** e veja o resultado.
